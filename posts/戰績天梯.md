## 📈 累計數字（showing up = 加分）

| 指標 | 累計 | 說明 |
|------|------|------|
| 連續實作週數 | `2` | 每個休息日有動手就 +1;斷了歸零 |
| push 次數 | `9` | = GitHub 綠格子數,免費客觀裁判 |
| 解題數 | `3` | 打通(本地或 remote)才算 |
| shell 彈出數 | `2` | 最硬的「喀一聲」,彈一次記一次 |
| 機制喀一聲數 | `5` | 某機制在腦中成形的次數(不需解題也算) |

> ⭐ 記「機制喀一聲數」是故意的:硬題一週彈不出來時,只要有一個機制懂了,這格照樣 +1 → 你就不會空手而歸。
